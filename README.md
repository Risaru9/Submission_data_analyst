# E-Commerce Analytics Dashboard

## Deskripsi Proyek
Proyek ini bertujuan untuk melakukan analisis data pada E-Commerce Public Dataset
guna memahami performa bisnis, perilaku pelanggan, serta distribusi geografis
pelanggan. Hasil analisis divisualisasikan dalam bentuk dashboard interaktif
menggunakan Streamlit.

## Dataset
Dataset yang digunakan merupakan E-Commerce Public Dataset yang terdiri dari
beberapa tabel seperti orders, order_items, customers, dan products. Tidak
seluruh tabel digunakan, pemilihan dataset dilakukan berdasarkan relevansi
terhadap tujuan analisis.

## Tahapan Analisis
1. Data Gathering
2. Data Cleaning dan Preprocessing
3. Exploratory Data Analysis (EDA)
4. RFM Analysis
5. Geospatial Analysis
6. Customer Clustering (Non-Machine Learning)
7. Insight dan Business Implication

## Dashboard
Dashboard dibuat menggunakan Streamlit dan menampilkan:
- Business Performance Overview
- Customer Intelligence (RFM & Clustering)
- Geographic Distribution of Customers
